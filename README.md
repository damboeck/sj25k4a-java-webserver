# Webserver Einführung
## Github Repository
https://github.com/damboeck/sj25k4a-java-webserver.git

## Allgemeines
Dieses Projekt stellt einen einfachen Webserver zur Verfügung, der auf Java und dem Spring Boot Framework basiert. Der Webserver kann verwendet werden, um statische Inhalte zu hosten und einfache RESTful APIs bereitzustellen.
